# 🌦️ WeatherNow - Weather App

WeatherNow is a modern and responsive weather application built with **HTML, CSS, and JavaScript**. It allows users to search for any city worldwide and instantly view real-time weather information using the **OpenWeatherMap API**.

The application provides essential weather metrics such as temperature, humidity, wind speed, visibility, sunrise/sunset timings, and more through a clean and user-friendly interface.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Display current temperature
* 🔄 Toggle between Celsius (°C) and Fahrenheit (°F)
* 🤗 View "Feels Like" temperature
* 💧 Check humidity levels
* 💨 View wind speed
* 👀 Display visibility distance
* 🌅 Show sunrise time
* 🌇 Show sunset time
* 📊 Air pressure indicator with visual pressure bar
* ⏳ Loading state while fetching weather data
* ❌ Error handling for invalid or unavailable cities
* 📱 Responsive design for different screen sizes

## 📖 How to Use

1. Enter a city name in the search field.
2. Press **Enter** or click the **Search** button.
3. Wait for the weather data to load.
4. View detailed weather information for the selected city.
5. Toggle between Celsius and Fahrenheit if needed.

---

## 🌐 API Information

This project uses the **OpenWeatherMap Current Weather API** to retrieve real-time weather data.

Official API Website:

https://openweathermap.org/

To use the application, you must create a free OpenWeatherMap account and generate an API key.
